# Token Logos

Place your token logo images here.

## Required Files:
- `FMY.png` - Fairmoney Microfinance Bank
- `ABMFB.png` - AB Microfinance Bank
- `FCMB.png` - First City Monument Bank
- `NOMBA.png` - Nomba MFB
- `OPAY.png` - Opay Digital Service

## Image Requirements:
- Format: PNG, JPG, or SVG
- Recommended size: 128x128px or larger (square)
- Transparent background preferred
- File names must match token symbols exactly (case-sensitive)

## After Adding Logos:
Run this command to update the database:
```bash
npx tsx scripts/update-logos-local.ts
```
